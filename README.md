# @zivego/n8n-nodes-dfir-iris

DFIR IRIS node integration for [n8n](https://n8n.io/).

Repository: `zivego/dfir-iris-n8n-node`

## Install

Install from the `n8n` UI.

For the latest version, enter:

```text
@zivego/n8n-nodes-dfir-iris
```
<div align="center">
  <img width="650" height="471" alt="555" src="https://github.com/user-attachments/assets/242035af-d494-4def-b66e-8a18be78473e" />
</div>
If you need an older version, enter it with the npm version suffix, for example:

```text
@zivego/n8n-nodes-dfir-iris@4.0.0
```

After installation, select:

- node: `DFIR IRIS (Zivego)`
- credentials: `DFIR IRIS API (Zivego)`
